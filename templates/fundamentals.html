What is complexity analysis?

We have this notion of complexity analysis - a way to identify the tradeoffs between various algorithms, both in terms of the amount of time that an algorithm will take to solve a problem but also the amount of space that it will require in memory. We can say that for a growing input size n, the amount of time an algo necessitates can be articulated as a function of n itself, say, f(n). This is useful when comparing the amount of Time that a given algo necessitates for a changing Input Size.
